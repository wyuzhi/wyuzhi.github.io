git using

发表于 2017-12-26

·创建一个空文件夹，”git init” 初始化仓库；
·文件夹中创建一个文本文件：”git add 名称.txt” 将文件添加到git仓库的缓存区；
(说明：git add 命令是将内容添加至缓存区，即可以添加多个文件，然后使用git commit命令 将多个文件同时提交)
·” git commit –m “对提交文件的说明” “:
(简单解释一下git commit命令，-m后面输入的是本次提交的说明，可以输入任意内容，当然最好是有意义的，这样你就能从历史记录里方便地找到改动记录。-廖雪峰)
·”git log” 查看日志
